# BITSUDOKU - Bit Board Sudoku Solver

# Build
- Requires a compiler that supports C++17 or higher.
- The default is C++20. (CMakeLists.txt)

```
$ cd bitsudoku
$ mkdir build
$ cd build
$ cmake ..
$ make
```

# bitsudoku

```
$ ./bitsudoku [3|4] sudoku_problem_file.txt
```

# The Problem File Format
- Numbers and spaces are separated by spaces'` `'.
- The numbers are decimals starting from 1.
- Space is '`-`'.
